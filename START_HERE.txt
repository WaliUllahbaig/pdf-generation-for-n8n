1️⃣  POST /html-to-pdf (Main Endpoint)
    Convert HTML string to PDF
    Input: JSON {"html": "<html>...</html>"}
    Output: Binary PDF file
    Errors: 400 (validation), 500 (generation)

2️⃣  GET /health (Health Check)
    Docker/Kubernetes health monitoring
    Output: {"status": "ok", ...}
    Use for: Orchestration health checks

3️⃣  GET /info (API Information)
    Get API capabilities & metadata
    Output: Full API information
    Use for: Client discovery

Problem: "Module not found"
Solution: pip install -r requirements.txt

Problem: "Cannot connect to http://localhost:5001"
Solution: Make sure API is running (python app.py or docker-compose up -d)

Problem: "Port 5001 in use"
Solution: Change port in app.py or docker-compose.yml

Problem: "Conda not recognized"
Solution: Use "Anaconda Prompt" instead of regular Command Prompt

Problem: "Docker not found"
Solution: Install Docker Desktop or use local Python setup
